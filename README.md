# 致癌沙拉油．發票自動比對

上傳財政部電子發票（手機條碼載具）消費明細 CSV，在**瀏覽器內**自動比對「中聯／苯駢芘沙拉油」事件的受影響產品，把可疑購買整理出來並可匯出 CSV。全程不上傳任何資料。

## 使用

打開網頁 → 上傳一或多個電子發票 CSV → 開始比對 → 下載比對結果 CSV。

## 內建對照表（擷取 2026-07-08）

- **受影響油品**：食藥署「中聯油脂案專區」受影響油品資訊（含批號／有效日期）
- **受影響成品**：食藥署「預防性下架產品清單」401 項官方公告（含業者／有效日期），另有少數商周懶人包補充項
- 完整清單見 [`問題產品清單.csv`](問題產品清單.csv)，網頁內也可展開／下載

## 說明

- 以「商品名稱關鍵字」比對，只能找出**可疑候選**；真正判定依據是**產品批號／有效日期**，發票明細通常不含批號，請以食藥署最新公告與各業者客服為準。
- 名單為快照，最新版請見 [食藥署中聯油脂案專區](https://www.fda.gov.tw/tc/siteList.aspx?sid=13708)。
